## üß† How It Works

1. A timestamped session folder is created at startup.
2. When monitoring begins:
   - Keystrokes typed inside the app are logged.
   - A separate thread captures screenshots periodically.
3. Both logs and images are stored using timestamps for correlation.
4. Monitoring stops when the user clicks "Stop Monitoring."
